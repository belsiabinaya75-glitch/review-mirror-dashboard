# Review Mirror Dashboard

## Project Overview

Review Mirror Dashboard is a Power BI project developed to analyze tourist reviews and ratings. The dashboard provides insights into visitor feedback, rating distributions, and sentiment trends through interactive visualizations, helping to better understand tourist experiences and opinions.

## Features

* Tourist review trend analysis
* Rating distribution visualization
* Sentiment analysis of visitor feedback
* Interactive filters and slicers
* Review pattern exploration through visual dashboards

## Tools Used

* Power BI
* Excel / CSV
* Data Modeling
* DAX

## Key Insights

* Identified rating patterns and distributions.
* Analyzed positive and negative review trends.
* Explored visitor feedback behavior using interactive visualizations.
* Generated insights from real-world review data.

## Project Outcome

The dashboard transforms tourist review data into meaningful insights, enabling a better understanding of visitor experiences, sentiment trends, and overall feedback patterns.
